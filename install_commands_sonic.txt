0) make sure kernel headers of current version are installed: e.g.
sudo apt install linux-headers-5.13.0-51-generic

1) get driver at https://www.asix.com.tw/en/support/download
2) compile
tar -jxvf AX99100_SP_PP_SPI_Linux_Driver_v1.8.0_Source.tar.bz2 
cd AX99100_SP_PP_SPI_Linux_Driver_v1.8.0_Source/
make
sudo insmod ax99100.ko

3) check with:
dmesg

4) ignore the make install, install with
sudo cp ax99100.ko /lib/modules/`uname -r`/kernel/drivers/tty/serial
sudo depmod

5) reboot 
6) check 
lsmod
ls /dev/ttyF0


7) enable autostart of the script
ln -s /home/visss/VISSS/scripts/sonic_logger.desktop /home/visss/.config/autostart/

add cronjob for gzip cpmpression

2 1 * * * python3 /home/visss/VISSS/sonic/sonic_gzip_files.py
